# Campus Notifications Priority Inbox

A production-grade frontend application for the 
Affordmed Campus Hiring Evaluation.

## Repository Structure

notification_app_fe/        — Stage 1: Priority Inbox
notification_app_stage2/    — Stage 2: Full Notifications App
logging_middleware/         — Reusable logging middleware
notification_app_be/        — Backend (not implemented)
notification_system_design.md — Architecture document

## Tech Stack

- React with TypeScript
- Material UI
- Custom logging middleware

## Running Stage 1

cd notification_app_fe
npm install
npm start

## Running Stage 2

cd notification_app_stage2
npm install
npm start

## Notes

- Both apps run on http://localhost:3000
- Enter your credentials in the login form
- No environment files needed
