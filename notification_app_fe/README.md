# Campus Notifications — Stage 1

Priority Inbox frontend application.

## Features
- Authentication with clientID and clientSecret
- Priority-ranked notifications (Placement > Result > Event)
- Category filtering (All / Placement / Result / Event)
- Top N selector (5, 10, 20, 50)
- Responsive design (mobile and desktop)
- Production-grade logging middleware

## Setup
npm install
npm start
