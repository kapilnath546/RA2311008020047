# Campus Notifications — Stage 2

Full notifications application with priority inbox.

## Features
- Authentication with clientID and clientSecret
- All Notifications page with pagination and filtering
- Priority Notifications page with heap-based ranking
- Read/Unread notification tracking
- Category filtering using API query parameters
- Top N selector (5, 10, 20, 50)
- Responsive design (mobile and desktop)
- Production-grade logging middleware

## Setup
npm install
npm start

## Pages
- /notifications — All notifications with filter and pagination
- /priority — Priority ranked top N notifications
